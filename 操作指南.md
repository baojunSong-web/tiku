# 孩子的错题库系统 - 操作指南

## 🚀 5分钟快速启动指南

### 准备工作
1. **下载并安装 Node.js**
   - 打开 [Node.js 官网](https://nodejs.org/zh-cn/)
   - 点击 "下载" 按钮，选择 LTS 版本
   - 双击安装包，按照提示完成安装
   - 安装完成后，关闭并重新打开所有浏览器窗口

2. **下载项目文件**
   - 从提供的链接下载项目压缩包
   - 解压到电脑上的任意位置（建议放在桌面）

### 启动程序

#### 步骤1：启动后端服务
1. 打开项目文件夹，找到 `backend` 文件夹
2. 在 `backend` 文件夹空白处，按住 **Shift** 键，同时右键点击
3. 选择 "在此处打开命令窗口" 或 "在此处打开 PowerShell 窗口"
4. 在弹出的黑色窗口中输入：`npm install`，然后按回车键
5. 等待安装完成（可能需要2-5分钟）
6. 安装完成后，输入：`npm run dev`，然后按回车键
7. 看到 "服务器运行在 http://localhost:3001" 后，后端服务启动成功

#### 步骤2：启动前端服务
1. 保持后端服务的黑色窗口打开
2. 打开项目文件夹，找到 `frontend` 文件夹
3. 在 `frontend` 文件夹空白处，按住 **Shift** 键，同时右键点击
4. 选择 "在此处打开命令窗口" 或 "在此处打开 PowerShell 窗口"
5. 在弹出的黑色窗口中输入：`npm install`，然后按回车键
6. 等待安装完成（可能需要2-5分钟）
7. 安装完成后，输入：`npm run dev`，然后按回车键
8. 浏览器会自动打开，显示 "错题列表" 页面

### 开始使用
1. 点击 "添加错题" 按钮，开始录入第一道错题
2. 填写题目内容、答案等信息
3. 点击 "保存" 完成录入
4. 查看错题列表，体验各种功能

## 📖 完整操作指南

### 一、系统介绍

**孩子的错题库系统** 是一个帮助您管理孩子错题的工具，可以：
- ✅ 手动录入错题
- ✅ 拍照/截图识别错题
- ✅ 按科目、知识点分类管理
- ✅ 生成相关练习题
- ✅ 打印练习试卷
- ✅ 标记重点题目
- ✅ 制定复习计划

### 二、安装与启动

#### 1. 安装 Node.js
Node.js 是运行这个系统的必要软件，安装步骤：

1. 打开 [Node.js 官网](https://nodejs.org/zh-cn/)
2. 点击 "下载" 按钮，选择 **LTS 版本**（长期支持版）
3. 双击下载的安装包，按照提示完成安装
4. 安装完成后，重启所有浏览器窗口

#### 2. 启动系统

**注意：** 系统分为两个部分，需要分别启动

##### 启动后端服务
1. 打开项目文件夹，找到 `backend` 文件夹
2. 在文件夹空白处，按住 **Shift** 键 + 右键点击
3. 选择 "在此处打开命令窗口" 或 "在此处打开 PowerShell 窗口"
4. 输入：`npm install`，按回车键
5. 等待安装完成（屏幕上会显示进度）
6. 输入：`npm run dev`，按回车键
7. 看到 "服务器运行在 http://localhost:3001" 后，后端服务启动成功
8. **不要关闭** 这个黑色窗口

##### 启动前端服务
1. 保持后端服务窗口打开
2. 找到 `frontend` 文件夹，重复步骤 2-3
3. 输入：`npm install`，按回车键
4. 等待安装完成
5. 输入：`npm run dev`，按回车键
6. 浏览器会自动打开，显示系统主页面

#### 3. 访问系统
- 浏览器地址栏显示：http://localhost:5173 或类似地址
- 页面显示 "错题列表"，说明系统启动成功

### 三、功能使用说明

#### 1. 添加错题

##### 手动录入
1. 点击页面右上角的 "添加错题" 按钮
2. 选择科目（语文、数学、英语）
3. 输入题目内容
4. 输入正确答案
5. 输入孩子的错误答案（可选）
6. 输入错误原因（可选）
7. 输入题目解析（可选）
8. 设置难度级别（1-5）
9. 选择相关知识点和错误类型（可选）
10. 点击 "保存" 按钮

##### 图片识别录入
1. 点击页面右上角的 "添加错题" 按钮
2. 选择 "图片上传" 选项卡
3. 点击 "选择图片" 按钮，选择包含错题的图片
4. 系统会自动识别图片中的文字
5. 检查并修改识别结果
6. 填写其他信息（科目、难度等）
7. 点击 "保存" 按钮

#### 2. 管理错题

##### 查看错题列表
- 首页显示所有错题
- 可以按科目、知识点、时间等筛选
- 可以标记题目为 "收藏"

##### 编辑错题
1. 找到要编辑的错题卡片
2. 点击卡片右上角的 "编辑" 按钮
3. 修改题目信息
4. 点击 "保存" 按钮

##### 删除错题
1. 找到要删除的错题卡片
2. 点击卡片右上角的 "删除" 按钮
3. 在弹出的提示框中点击 "确定"

##### 批量删除
1. 勾选多个错题卡片左上角的复选框
2. 点击页面顶部的 "删除选中题目" 按钮
3. 在弹出的提示框中点击 "确定"

#### 3. 生成相关练习题

1. 找到一道错题，点击卡片右上角的 "生成相关题" 按钮
2. 在弹出的窗口中，设置：
   - 生成数量：1-10 道
   - 难度级别：简单到非常困难
3. 点击 "生成相关题目" 按钮
4. 等待几秒，系统会生成相关练习题
5. 可以查看生成的题目，或点击 "生成 PDF" 保存为练习试卷

#### 4. 打印错题

##### 打印选中的错题
1. 勾选要打印的错题卡片
2. 点击页面顶部的 "打印选中题目"
3. 在预览窗口中查看效果
4. 点击 "生成 PDF" 按钮，下载 PDF 文件
5. 使用 PDF 阅读器打开并打印

##### 打印生成的练习题
1. 生成相关练习题后，点击 "生成 PDF" 按钮
2. 在预览窗口中查看效果
3. 点击 "生成 PDF" 按钮，下载 PDF 文件
4. 使用 PDF 阅读器打开并打印

#### 5. 复习管理

##### 标记重点题目
1. 找到要标记的错题卡片
2. 点击卡片右上角的心形图标
3. 图标变红表示已收藏

##### 筛选重点题目
1. 点击左侧筛选面板
2. 勾选 "只看收藏题目"
3. 页面只显示已收藏的重点题目

### 四、常见问题及解决方法

#### 1. 浏览器无法打开系统
- **原因**：服务没有正确启动
- **解决方法**：
  - 检查后端和前端的黑色窗口是否都在运行
  - 检查窗口中是否有错误信息
  - 重新启动后端和前端服务

#### 2. 添加错题时无法保存
- **原因**：必填项没有填写完整
- **解决方法**：
  - 检查是否填写了题目内容和正确答案
  - 检查科目是否选择
  - 检查网络连接是否正常

#### 3. 图片识别不准确
- **原因**：图片质量不好或文字太小
- **解决方法**：
  - 确保图片清晰，文字大小适中
  - 手动修改识别结果
  - 尝试重新上传图片

#### 4. 生成练习题失败
- **原因**：DeepSeek API 配置问题
- **解决方法**：
  - 检查 `frontend/.env` 文件中的 API 密钥是否正确
  - 确保网络连接正常
  - 稍后再试

#### 5. 端口被占用
- **原因**：其他程序正在使用相同的端口
- **解决方法**：
  - 关闭其他可能占用端口的程序
  - 重启电脑
  - 重新启动系统服务

### 五、系统维护

#### 1. 备份数据
- 系统数据保存在 `database` 文件夹中
- 定期复制 `tiku.db` 文件到其他位置，作为备份

#### 2. 关闭系统
- 关闭所有黑色命令窗口
- 关闭浏览器窗口

#### 3. 重新启动系统
- 按照 "启动系统" 部分的步骤重新启动

### 六、技术支持

如果遇到无法解决的问题，可以：
1. 查看系统日志（黑色命令窗口中的信息）
2. 检查网络连接
3. 重新安装 Node.js
4. 重新下载项目文件

## 📝 使用技巧

1. **定期整理错题**：每周花10分钟整理一周的错题
2. **合理使用生成题**：为重点错题生成相关练习题，加深理解
3. **标记重点题目**：使用收藏功能标记需要重点复习的题目
4. **按科目管理**：不同科目的错题分开管理，便于针对性复习
5. **定期打印练习**：将错题和生成题打印出来，进行纸质练习

## 🎯 最佳实践

1. **每天录入**：当天的错题当天录入，避免忘记
2. **详细记录**：尽量填写错误原因和解析，便于后续复习
3. **分类管理**：使用知识点和错误类型标签，便于筛选
4. **定期复习**：每周复习一次收藏的重点错题
5. **生成练习**：为每道重点错题生成3-5道相关练习题
